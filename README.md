 ## 📝 Game Hub
*Este projeto consiste em uma plataforma integrada de jogos clássicos executados via terminal. O objetivo principal é consolidar conceitos de Programação Orientada a Objetos (POO) e a organização de projetos em Packages e Módulos no Python.*
## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.13
* **IDE:** PyCharm
* **Bibliotecas:** 
    * `colorama`: Para estilização de cores no terminal.
    * `os`: Para manipulação de comandos do sistema (limpeza de tela).
    * `Random` : Para sortear resultados aleátorios.
## ✨ Funcionalidades
* **Menu Interativo:** Navegação centralizada entre diferentes jogos através de `match-case`.
* **Jokempo:** Modo singleplayer contra a máquina.
